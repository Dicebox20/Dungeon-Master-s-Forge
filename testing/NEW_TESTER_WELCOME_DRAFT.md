# New Tester Welcome Draft

Welcome to the Dungeon Master's Forge tester group, and thank you for helping us improve it.

The current public tester build is **2.23.1-test.63**. Install it in Foundry from this manifest URL:

`https://raw.githubusercontent.com/Dicebox20/Dungeon-Master-s-Forge/dm_forge/tester/testing/module.json`

For your first session:

1. Enable **Dungeon Master's Forge** in a test world.
2. Leave the provider set to **Free Forge** and use **Check Connection**.
3. Try one simple item first, then a charged, healing, or multi-activity item.
4. Read the preview and every review note before approving anything.
5. Open created items in Foundry and test the relevant activity safely.

If the result is close but a mechanic is wrong or missing, leave the original prompt unchanged and click **Retry**. In **SEND IT AGAIN!?**, describe only the correction in **What should it have done?**, check **I understand**, and click **Send repair request** once. The repaired result returns for fresh review; it does not create or execute anything automatically.

Please tell us what you expected, what Foundry actually showed, and whether the problem appeared in the preview, review notes, created item, or activity behavior. A visible error message, request ID, immediate structured snapshot, or exported item JSON can help when available. Never include API keys, passwords, or private campaign information.

Thank you for testing carefully. A result that needs manual review is useful feedback, not a failed test by itself.
